## Lists

Here's plain old bullet list:

1. Item 1

You can even do checklists:

- [x] Learn programming

- Foods I like

  - Domino's Pizza

## Code

You can inline code like `this`.

You can also create code blocks:

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(f"Fibonacci(10) = {fibonacci(10)}")
```
