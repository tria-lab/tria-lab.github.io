## Setting up development environment

1. Install [vscode](https://code.visualstudio.com/).
   - vscode will prompt you to install required extensions when you open this repository.
2. Setup [mise-en-place](https://mise.jdx.dev/getting-started.html).
   - you may instead install the contents of [`mise.toml`](./mise.toml) manually but the behavior
     may differ from production.
3. Launch dev server and start coding.
   ```
   bun run dev
   ```
